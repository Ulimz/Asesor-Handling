# Asistente_Handling Backend Overview

Backend modular para asistente legal en el sector handling aeroportuario español.

## Estructura inicial
- FastAPI
- PostgreSQL
- Elasticsearch
- Docker
- CI/CD

## Módulos
- usuarios
- empresas
- convenios
- artículos
- jurisprudencia
- reclamaciones
- calculadoras
- alertas
- ia

## Instalación y uso
1. Instala dependencias: `pip install -r requirements.txt`
2. Ejecuta el servidor: `uvicorn main:app --reload`

## Documentación automática
Accede a `/docs` para la API Swagger.
